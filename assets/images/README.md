# Images Directory

Place your image files here:

## Required Images

1. **Logo** (`logo.svg` or `logo.png`)
   - Place your logo file here
   - Recommended: SVG format for scalability
   - Update the logo path in `index.html` if using a different filename

2. **Open Graph Image** (`og-image.jpg`)
   - Social media preview image
   - Recommended size: 1200x630px
   - Update the og:image meta tag in `index.html` with your actual URL

3. **Event Photos** (for gallery)
   - Replace the placeholder images in the gallery section
   - Recommended format: JPG or WebP
   - Recommended size: 800x800px minimum
   - Update image src attributes in `index.html`

4. **Announcement Images**
   - Replace placeholder images in announcement cards
   - Recommended size: 800x600px
   - Update image src attributes in `index.html`

5. **Instagram Preview Images**
   - Replace placeholder images in Instagram section
   - Recommended size: 400x400px
   - Update image src attributes in `index.html`

## Current Placeholder Images

The website currently uses Unsplash placeholder images for demonstration purposes. These should be replaced with your actual images before going live.

## Image Optimization Tips

- Use WebP format when possible for better compression
- Compress images to reduce file size (use tools like TinyPNG or ImageOptim)
- Use descriptive alt text for accessibility
- Ensure images are properly sized to avoid layout shifts
